# Arena Shooter 3D

A 3D first-person shooter game built with Three.js where you battle against an AI opponent in an arena.

## How to Play

1. Open `index.html` in your web browser
2. Use **Arrow Keys** to move around the arena
3. Press **Space Bar** to shoot
4. Defeat the enemy (red character) before they defeat you!

## Game Features

- 🎮 First-person 3D perspective
- 🤖 AI-controlled enemy opponent
- 🎯 Shooting mechanics with cooldown
- 🏗️ Arena with obstacles for cover
- ❤️ Health system for both player and enemy
- 📊 Score tracking
- 🔄 Restart capability

## Controls

- **Arrow Up**: Move forward
- **Arrow Down**: Move backward
- **Arrow Left**: Move left
- **Arrow Right**: Move right
- **Space**: Shoot

## Gameplay

- You (blue) start at one end of the arena
- Enemy (red) starts at the opposite end
- Hit the enemy 10 times to win
- Avoid getting hit 10 times or you lose
- Use obstacles for cover
- Enemy AI moves randomly and shoots at you

## Running the Game

Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

No installation or build process required!
